To build the examples, use the following libraries (for Win32/x86).

libSDL2.dll.a
libSDL2_image.dll.a
libSDL2_mixer.dll.a
libcurldll.a

To run the examples, put the following libraries here (for Win32/x86).

libcurl.dll
libjpeg-9.dll
libpng16-16.dll
SDL2.dll
SDL2_image.dll
zlib1.dll
