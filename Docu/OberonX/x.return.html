OberonX, Structured Return Values

OberonX allows procedures to return any type. This should allow a better abstraction in the language and help in the implementation of mathematical code, making the use of functional code much simpler.

Example: 

 
TYPE 
     Vector = ARRAY 3 OF LONGREAL;


PROCEDURE MakeVector(a0, a1, a2: LONGREAL): Vector; 
     VAR v: Vector; 
     BEGIN 
          v[0] := a0; v[1] := a1; v[2] := a2; 
          RETURN v 
     END MakeVector;


PROCEDURE Add(a, b: Vector): Vector; 
     BEGIN 
          a[0] := a[0] + b[0]; 
          a[1] := a[1] + b[1]; 
          a[2] := a[2] + b[2]; 
          RETURN a 
     END Add;
