To build the examples, use the following libraries (for Win64/x64).

libSDL2.dll.a
libSDL2_image.dll.a
libSDL2_mixer.dll.a
libcurldll.a

To run the examples, put the following libraries here (for Win64/x64).

libcrypto-1_1-x64.dll
libcurl-x64.dll
libjpeg-9.dll
libpng16-16.dll
libssl-1_1-x64.dll
msvcr110.dll
SDL2.dll
SDL2_image.dll
zlib1.dll
